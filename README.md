# 🔭 Radar de Fondos Concursables

> Navega y descubre fondos para educación, patrimonio, desarrollo rural (agro) y conservación ambiental, con foco especial en Ñuble y San Fabián de Alico.

![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white)
![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-222222?style=for-the-badge&logo=githubpages&logoColor=white)

## ✨ Características

- 🔍 **Búsqueda en tiempo real** — Filtra fondos por nombre, organización o palabra clave
- 🏷️ **Categorías inteligentes** — Gobierno, Patrimonio, Educación, Agrupación Social, Agro y Rural, Conservación y Medioambiente
- 🎨 **Diseño moderno glassmorphism** — Interfaz oscura con efectos de vidrio y gradientes
- 📱 **Responsivo** — Se adapta a cualquier dispositivo
- ⚡ **Rápido** — Construido con Vite y TypeScript

## 🗂️ Categorías de Fondos

| Categoría | Descripción |
|-----------|-------------|
| 🏛️ Gobierno | FNDR, Subvenciones Presidenciales |
| 🏺 Patrimonio | Patrimonio Cultural, Restauración |
| 📚 Educación | FAE, Infraestructura Escolar, CEIA |
| 🤝 Agrupación Social | FONDEPORTE, Desarrollo Comunitario |
| 🌾 Agro y Rural | INDAP, FIA, Innovación Agraria |
| 🌿 Conservación | CONAF, Bosque Nativo |

## 🚀 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/<tu-usuario>/radar-fondos.git
cd radar-fondos

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## 🔧 Scripts

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Servidor de desarrollo en localhost |
| `npm run build` | Compilar para producción |
| `npm run preview` | Previsualizar build de producción |

## 📦 Deploy

Este proyecto se despliega automáticamente en **GitHub Pages** mediante GitHub Actions cada vez que se hace push a la rama `main`.

## 🛠️ Tech Stack

- **Frontend**: Vite + TypeScript + Vanilla CSS
- **Diseño**: Glassmorphism, Google Fonts (Outfit + Inter)
- **Deploy**: GitHub Pages + GitHub Actions
